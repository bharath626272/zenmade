# Zenmade

This project now runs as a React frontend application.

## Run locally

1. Open the frontend folder.
2. Install dependencies with npm:
   npm install --legacy-peer-deps
3. Start the app:
   npm start

The app is available at http://localhost:3000.

## Notes

- The old Python backend folder is no longer used for the main app flow.
- The contact form now works as a frontend-only interaction.
